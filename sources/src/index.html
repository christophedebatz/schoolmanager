Bienvenue dans School Manager

School Manager est un logiciel tout-en-un de gestion d'école. Il est divisé en deux parties distinctes :

  • Le mode serveur qui permet à chacun de créer son propre serveur sur lequel d'autres utilisateurs pourront se connecter afin de modifier la base de données de School Manager.
  • Le mode client permettent à chacun de se connecter à un serveur, sous réserve de posséder l'adresse IP de celui-ci ainsi que le port de connexion.

    • School Manager est un programme intuitif et clair. Vous ne mettrez que quelques secondes avant de comprendre comment fonctionne le tout.

    Voici les étapes pour vous guider :

    1. Cliquez sur la deuxième icône de la barre de menu pour vous connecter à un serveur 2. Cliquez sur la troisième, quatrième et cinquième icône de la barre de menu pour respectivement, accéder à la gestion des cours, des professeurs ainsi que des élèves.
